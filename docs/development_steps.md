# 开发步骤

## 一、算法核心闭环

- 完成波形生成、工程指标、匹配滤波、零多普勒旁瓣、二维模糊函数、探测性能、宽带噪声压制干扰、LPI 暴露特征和分辨能力指标。
- 建立 `compute_waveform_evaluation` 评估流水线，统一输出 raw metrics、axis scores、overall score 和轻量 chart data。
- 使用配置化评分，明确评分依赖归一化边界和权重，不把默认配置解释为绝对物理标准。
- 使用 CLI 读取配置并输出 JSON 文件，形成可重复运行的算法闭环。

## 二、桌面软件闭环

- 在 PySide6 / QtWidgets 中接入算法评估流水线，UI 只负责输入、展示和任务调度，不写雷达公式。
- 完成波形配置、场景配置、评估参数、指标表、雷达图和曲线图的基础交互。
- 增加后台 worker，避免长时间评估阻塞主界面。
- 增加项目文件保存、加载和结果对比。

## 三、报告、打包和交付闭环

- 在结构化指标结果基础上生成报告，不让大语言模型直接计算雷达指标。
- 增加 CSV、图片、PDF 或 Word 等导出能力。
- 完善 PyInstaller 打包脚本和运行环境验证。
- 补充交付文档、错误排查说明和示例配置。
