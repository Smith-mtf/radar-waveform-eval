# 开发步骤

## 阶段一：工程骨架

- 建立 Python 项目配置。
- 建立 `radar_eval_core` 和 `desktop_app` 边界。
- 建立基础数据结构和占位算法接口。
- 增加最小 PySide6 主窗口。
- 增加导入测试和数据结构测试。

## 阶段二：核心指标

- 逐步实现波形生成、匹配滤波、模糊函数、检测概率等核心能力。
- 每个公共算法函数配套 pytest 测试。
- 使用固定输入和可解释的基准结果验证算法。

## 阶段三：桌面交互

- 增加页面切换。
- 增加波形、场景、干扰配置表单。
- 增加图表、指标表和结果对比。
- 增加后台评估任务，避免阻塞 UI。

## 阶段四：导出与打包

- 增加项目文件保存和加载。
- 增加 CSV、图片和报告导出。
- 使用 PyInstaller 建立打包脚本。
