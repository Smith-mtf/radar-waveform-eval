# 指标体系 V1.0

V1.0 只定义指标分组和数据结构，不直接实现复杂公式。

## 指标分组

- 探测性能：用于描述波形在指定场景中的目标探测能力。
- 抗干扰性能：用于描述波形面对压制、欺骗、扫频等干扰时的性能保持能力。
- 反侦察性能：用于描述波形被截获、识别和参数估计的风险水平。
- 工程可实现性：用于描述带宽、时宽、采样率、功率、复杂度等工程约束。

## 结果表达

每个指标使用 `MetricValue` 表达，包含指标名称、数值、单位和说明。每个评估维度使用 `AxisScore` 表达，包含维度名称、得分和指标列表。完整评估使用 `EvaluationResult` 表达。

## 约束

- 指标计算必须在 `radar_eval_core` 中实现。
- UI 层只展示和组织结构化结果。
- 大语言模型只能解释结构化指标结果，不允许直接计算雷达指标。
