# 模型假设 V1.0

## 波形模型

当前支持复基带 `rect`、`lfm` 和 `phase_code` 波形。IQ 幅度按 `sqrt(W)` 标定时，`abs(iq) ** 2` 可解释为功率，单位为 W。

## 探测模型

探测性能使用单脉冲匹配滤波平方律检测模型。噪声为复高斯白噪声，目标为非起伏确定幅度、未知相位模型。当前不包含 CFAR、Swerling 起伏、多脉冲积累、复杂杂波或 Monte Carlo 检测曲线。

## 干扰模型

抗干扰模块只实现宽带复高斯噪声压制干扰。干扰被视为与热噪声独立的额外白噪声。当前不包含窄带干扰、扫频干扰、欺骗干扰、干扰识别或干扰抑制算法。

## 反侦察 / 低截获特征模型

LPI 模块只输出由波形本身可计算的暴露特征，例如峰值功率、平均功率、PAPR、名义平均 PSD、periodogram PSD、占用带宽、TBP 和占空比。当前不计算截获概率、截获距离比、侦收机检测门限或侦收机所需 SNR。

## 分辨能力模型

分辨能力包含距离分辨率、距离采样间隔、波长、CPI、多普勒分辨率和速度分辨率。CPI 必须直接给出，或由 `num_pulses + prf_hz` / `num_pulses + pri_s` 严格推导。

## 评分模型

评分依赖 `ScoringConfig` 中的归一化边界和权重。评分结果不是绝对物理结论，应结合评分配置解释。

## 未实现模型

V1.0 不实现：

- 截获概率
- 截获距离比
- CFAR
- Swerling I/II/III/IV
- 二维 PSLR / ISLR
- 复杂杂波
- 窄带、扫频和欺骗干扰
- 外部 LLM 自动报告
